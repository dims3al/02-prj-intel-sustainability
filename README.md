# Intel Sustainability Through the Ages

This project is a single-page timeline that highlights key Intel milestones and sustainability goals. It uses a horizontal scrollable card layout on desktop, plus a vertical stack on smaller screens.

## What's Included
- A hero header with the Intel logo and page title.
- Timeline cards with years, titles, and images.
- Intro and outro helper cards with always-visible text.
- Scroll snap and hover effects for a smooth, interactive experience.
- Responsive design for desktop and mobile.

## Files to Know
- [index.html](index.html) holds the page structure and timeline content.
- [style.css](style.css) controls layout, colors, and responsive behavior.
- [img](img) contains the images used in the cards.

## Notes
- The timeline uses flexbox for layout.
- Cards snap as you scroll on larger screens.
- On smaller screens, cards stack vertically for easy reading.
